## Getting Started

> CLI tool to get Alexa traffic ranks

### Requirements

* [Node.js](https://github.com/nodejs/node) - JavaScript runtime.

### Installation

```bash
npm i -g alexa-cli
```

### Usage

```bash
alexa <website>
```
